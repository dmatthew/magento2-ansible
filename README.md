# magento2-ansible
Vagrant VM provisioned with Ansible for use with Magento 2.

## Roles

### Server
* System settings and packages.

### Composer
* Installs composer.

### MySQL
* Installs and configures MySQL.

### PHP
* Installs PHP
  * Choose which version of PHP to install (5.5, 5.6, 7.0)

### NGINX
* Installs and configures NGINX.
